# 破冰小程序部署到苹果手机

## 当前可直接运行的形态

这个项目已经整理成适合 iPhone 使用的 PWA（渐进式网页应用）。
它不是原生 `ipa` 安装包，但用户可以在 Safari 里打开后“添加到主屏幕”，效果会接近普通 App。

## 你现在需要做的事

1. 把整个项目目录部署到一个静态网站上。
2. 用 iPhone 的 Safari 打开这个网址。
3. 点击 Safari 分享按钮。
4. 选择“添加到主屏幕”。
5. 主屏幕上就会出现“破冰小程序”图标，点击即可全屏运行。

## 推荐的部署方式

### 方式一：最省事

用静态托管平台部署：

- Cloudflare Pages
- Netlify
- Vercel
- GitHub Pages

把这个文件夹里的内容上传即可，首页文件是 `index.html`。

### 方式二：本地电脑临时预览

如果你只是自己手机测试，可以在电脑上启动一个本地静态服务器，然后让手机和电脑连接同一个 Wi‑Fi，通过电脑局域网地址访问。

注意：

- 录音功能在 iPhone 上通常要求 `https` 或安全上下文
- 所以正式使用时，最好部署到带 `https` 的网站

## iPhone 使用提醒

1. 首次使用录音功能时，Safari 会请求麦克风权限。
2. 浏览器可用音色受 iOS 系统限制，不同 iPhone 的系统 TTS 可能略有差异。
3. “我的录音腔”当前是播放用户录下的示范音，不是 AI 克隆成任意新文案。

## 如果你想变成真正的 iOS 原生 App

下一步建议这样迁移：

1. 用 Capacitor 把这个网页项目包进 iOS 壳子。
2. 在 Mac 上安装 Xcode。
3. 用 Xcode 打开 iOS 工程并真机调试。
4. 如果要上架，再配置 Apple Developer 证书和发布流程。
